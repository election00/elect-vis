# US Election Results Visualization

Interactive JavaScript/D3 visualization of US presidential election results.

Modes:
- State winner: colors each state by electoral winner.
- Partisan margin: shows how strongly each state leaned Democratic or Republican.
- Lean + vote total: maps partisan lean against total D/R vote amount.

Run locally with:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
